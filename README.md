# Tunneller

Tunneller is a macOS utility that automates login for VPN clients, including compatibility with Cisco Secure Client.

It lives in your menu bar and handles the full connection flow — entering your password and TOTP code — so you can connect with a single click.

## Features

- **One-click VPN connect** from the macOS menu bar
- **TOTP generation** built-in (RFC 6238, no external authenticator needed)
- **Credential sources**: macOS Keychain or 1Password CLI
- **AppleScript automation** of Cisco Secure Client's UI
- **Launch at login** support

## Requirements

- macOS 14+
- Cisco Secure Client installed
- Accessibility permission (System Settings → Privacy & Security → Accessibility)
